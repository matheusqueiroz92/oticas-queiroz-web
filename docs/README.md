# Documentação — Óticas Queiroz Web

- [ADRs](./adr/) — Architectural Decision Records, decisões importantes registradas

## Como contribuir com docs

- ADRs são imutáveis após aceitação. Para reverter, crie um novo ADR substituindo o anterior (use o campo `Status: Substituído por ADR-XXXX`).
- Runbooks vivem evoluindo. Atualize sempre que o procedimento mudar.
- Diagramas em Mermaid (preferido) ou imagens em `./assets/`.
